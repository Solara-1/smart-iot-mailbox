# Smart Mailbox IoT System
ESP32 Smart Mailbox with sensors, alert, servo lock, Blynk app control, and facial recognition.

## Overview
This project is a smart mailbox prototype built with an ESP32. It is designed to detect mailbox activity such as package delivery and mailbox access using sensor-based logic and actuator control.

## Hardware Used
- ESP32
- Ultrasonic sensor
- Pressure sensor
- Servo motor

## Features
- Detects delivery activity
- Detects mailbox opening/access events
- Uses embedded control logic for automated responses
- Supports IoT-style monitoring functionality

## Technologies
- Arduino C++
- Python
- ESP32
- Embedded systems
- Sensor integration

- `smart-mailbox.ino` — main mailbox control code
